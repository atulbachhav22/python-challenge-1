# Interactive ordering system

## Order System
Create an empty list. This list will later store a customer's order in dictionary format, as follows:
```sh
[
  {
    "Item name": "string",
    "Price": float,
    "Quantity": int
  },
  {
    "Item name": "string",
    "Price": float,
    "Quantity": int
  },
]
```
##Order Receipt
```sh
Item name                 | Price  | Quantity
--------------------------|--------|----------
Apple                     | $0.49  | 1
Tea - Thai iced           | $3.99  | 2
Fried banana              | $4.49  | 3
```
